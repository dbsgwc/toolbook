# ToolBook

> 个人命令速查小册：pm2、docker、nvm、git、pnpm 等高频工具。

- 快速复制命令（带示例）
- 高频场景：安装 / 常用 / 排障 / 最佳实践
- 支持站内本地搜索与暗黑模式

- 顶部导航或下方列表进入对应工具页

- 表格可直接复制命令；遇到问题看“常见场景/排障”
